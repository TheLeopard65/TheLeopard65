## 📜 Certification Overview

The **TryHackMe PT1 (Junior Penetration Tester)** certification is based on the popular **Jr Penetration Tester** learning path. It is a practical, beginner-friendly credential that validates core offensive security skills through hands-on labs and capture-the-flag (CTF) style challenges. The path covers the entire assessment workflow, from reconnaissance to privilege escalation.

## 🧠 Skills and Knowledge Acquired

### Reconnaissance

- Passive OSINT techniques (Google dorking, whois, social media).
- Active reconnaissance with Nmap (host discovery, port scanning, service enumeration).
- DNS enumeration (dig, nslookup, subdomain brute-forcing).

### Web Application Testing

- Exploiting common web vulnerabilities: file upload flaws, IDOR, command injection, XSS.
- Using Burp Suite for intercepting requests and fuzzing parameters.
- Directory busting with Gobuster and Dirb.

### Network Service Exploitation

- Enumerating and attacking SMB, FTP, SSH, and databases.
- Leveraging misconfigured services for initial access.
- Using Metasploit for auxiliary scanning and exploitation.

### Privilege Escalation

- Linux: SUID binaries, kernel exploits, cron jobs, and sudo rights.
- Windows: vulnerable services, registry misconfigurations, token impersonation.

### Scripting & Automation

- Basic Python and Bash scripting to automate tasks (port scanning, web requests).
- Understanding and modifying existing exploit code.

### Methodology & Mindset

- Approaching penetration testing systematically.
- Thinking like an attacker to find creative ways to compromise systems.
- Documenting steps and findings for future reference.
