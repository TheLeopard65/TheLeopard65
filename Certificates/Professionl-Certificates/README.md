## HTB Certified Penetration Testing Specialist (CPTS)

The CPTS is a 10 days (240 hours) hands-on certification from Hack The Box focused on performing full penetration tests in realistic environments. The exam involves working through a simulated network and demonstrating skills across Linux, Windows, web applications, and Active Directory.

Through this certification I gained experience with:

- Reconnaissance and information gathering (OSINT, passive and active scanning).
- Exploiting vulnerabilities in Linux and Windows environments.
- Web application testing for issues such as SQLi, XSS, SSRF, and IDOR.
- Working with common pentesting tools like Metasploit and Burp Suite.
- Post-exploitation tasks including privilege escalation, pivoting, and lateral movement.
- Writing clear penetration testing reports and documenting findings.

## INE Certified Professional Penetration Tester (eCPPT)

The eCPPT is a practical (48 hours) penetration testing certification from INE Security. The exam involves attacking a multi-segment network and producing a full penetration testing report at the end.

During this certification I worked on:

- Planning and executing penetration tests against complex network environments.
- Reconnaissance and vulnerability discovery using tools like Nmap and manual techniques.
- Exploiting common network and web vulnerabilities to gain initial access.
- Pivoting through segmented networks using tunneling and proxy techniques.
- Performing post-exploitation tasks such as credential harvesting and persistence.
- Documenting the full attack chain in a professional penetration testing report.

## INE Junior Penetration Tester (eJPT)

The eJPT is an entry-level practical certification focused on the fundamentals of penetration testing. The exam environment requires discovering and exploiting vulnerabilities across several systems.

Skills demonstrated during this certification include:

- Network scanning and service enumeration using Nmap.
- Identifying common vulnerabilities and misconfigurations.
- Exploiting services using tools such as Metasploit.
- Performing basic web attacks like directory traversal and simple SQL injection.
- Password attacks using tools like Hydra and John the Ripper.
- Working across Linux and Windows systems in a pentesting context.

## THM Junior Penetration Tester (PT1)

The TryHackMe PT1 certification is based on the **Jr Penetration Tester** learning path and focuses on foundational offensive security skills through hands-on labs and CTF-style challenges.

Topics covered include:

- Basic reconnaissance and OSINT techniques.
- Web application, Network and service enumeration.
- Exploiting common vulnerabilities in web applications and network services.
- Privilege escalation techniques on Linux and Windows.
- Using common tools such as Nmap, Gobuster, and John the Ripper.
- Developing a structured approach to penetration testing through practical labs.
