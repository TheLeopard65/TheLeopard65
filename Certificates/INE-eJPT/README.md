## 📜 Certification Overview

The **eJPT (Junior Penetration Tester)** is an entry-level, 100% practical certification from **INE Security**. It is designed to validate foundational penetration testing skills through a hands-on exam where candidates must assess a small network and achieve specific objectives. It serves as an excellent starting point for a career in offensive security.

## 🧠 Skills and Knowledge Acquired

### Network Scanning & Enumeration

- Using Nmap to discover live hosts, open ports, and running services.
- Banner grabbing and service version detection.
- Identifying potential entry points (e.g., open shares, weak configurations).

### Vulnerability Identification

- Manual and automated checks for common vulnerabilities (default credentials, misconfigurations).
- Basic web application assessment (directory traversal, parameter manipulation, simple SQLi).

### Exploitation

- Gaining initial access using Metasploit (auxiliary modules, exploits, payloads).
- Manual exploitation techniques for common services (FTP, SMB, SSH).

### Password Attacks

- Online brute-forcing with Hydra and Medusa.
- Offline hash cracking using John the Ripper and Hashcat.

### Post-Exploitation Fundamentals

- Basic privilege escalation on Linux and Windows (e.g., sudo -l, unquoted service paths).
- File transfers between systems (certutil, wget, SMB).
- Navigating and managing compromised systems.

### Methodology

- Following a structured pentesting process: information gathering, scanning, exploitation, and reporting.
- Documenting findings and steps taken during the assessment.
