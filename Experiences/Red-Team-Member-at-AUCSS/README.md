## 📜 Overview

As a **Red Team Core Member** at the Air University Cyber Security Society (AUCSS), I led hands-on penetration testing training, delivered technical seminars, and contributed to real-world attack simulations. My role focused on building practical offensive security skills within the student community, bridging the gap between academic knowledge and industry-grade adversarial techniques. I designed and delivered workshops on web application security, vulnerability assessment, and red team methodologies, while also producing actionable security reports based on simulated attacks.

## 🧠 Key Responsibilities & Contributions

- **Penetration Testing Training**

	- Led practical training sessions using dedicated "Pentest boxes" to teach Vulnerability Assessment and Penetration Testing (VAPT) concepts.
	- Covered web application security, network exploitation, and post-exploitation techniques.

- **Workshops on Modern Web Vulnerabilities**

	- Conducted workshops analyzing contemporary web vulnerabilities (e.g., OWASP Top 10, business logic flaws, API security).
	- Emphasized attack methodology, impact assessment, and defensive countermeasures.

- **Security Reporting & Mitigation**

	- Produced risk-driven penetration testing reports based on simulated attack scenarios.
	- Documented findings with clear business impact and provided actionable mitigation strategies.

- **Defense Strengthening**

	- Actively contributed to strengthening the society's internal security posture.
	- By identifying vulnerabilities in student-run platforms and suggesting improvements.
	- Promoted a culture of proactive security awareness among members.
